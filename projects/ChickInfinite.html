Presenting:
ChickInfinite

HackUCI 2018

GitHub Project Link

Languages/Technologies Used: Python, Kivy library

This project was a cross-platform game created by myself, Shayla Luong, and Alan Huang. In this game, the chicken at the top of the screen spawns eggs that fall across the screen. The player must tap on the eggs to gain points, and if an egg reaches the ground without being caught by the player, the player loses one life. The speed at which the eggs drop get progressively faster as the player gains points. This game is touch-screen compatible and can be played on mobile or desktop. Our game also received the award for Most Polished Game at HackUCI.

I personally worked on the programming the behavior of the eggs spawning and dropping across the screen, as well as the detection of screen taps or clicks from the player to properly despawn eggs and add to the score as the game progresses. I also implemented responsive design into our project for our game to be played at different screen sizing, allowing for both mobile and desktop gameplay.

This project holds a special place in my heart because it was the first project that I had created outside of assigned coursework. As this project was created while I was attending my first hackathon, I was still learning to manage the 36 hour time limit and create a project while collaborating in a group with the creative freedom provided by hackathons. Although this project was created during the first hackathon that I had ever attended, our group ended up winning the prize for Most Polished Game, which was very surprising to our entire group, and we were very proud of our accomplishment. The judges were impressed by the cross-platform compatability of our game, including the ability to play with either mouse, touchscreen laptop, or on mobile.

Overall, this was a very fun project that really opened my eyes to the world of hackathons, and from there I would continue to attend other hackathons and make other interesting projects.
